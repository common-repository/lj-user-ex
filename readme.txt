== Description ==

Replaces &lt;lj user="username"/&gt; and &lt;lj comm="community"/&gt; with correct HTML code. Like "LJ user" plugin, but supports both &lt;lj user="username"/&gt; and &lt;lj user="username"&gt; forms, plus communities.


== Installation ==

1. Upload `lj_user_ex.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
